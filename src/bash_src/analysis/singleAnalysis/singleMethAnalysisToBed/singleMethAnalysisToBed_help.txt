Description:
    This function makes BED files for the methylation analysis results obtained by means of 
    singleMethAnalysis followed by mergeSingleMethAnalysis:
    o MML-PHENO.bed: mean methylation levels
    o NME-PHENO.bed: normalized methylation entropy
    o METH-PHENO.bed: methylation-based classification (non-variable)
    o VAR-PHENO.bed: methylation-based classification (variable)
    o ENTR-PHENO.bed: entropy-based classification
    o ESI-PHENO.bed (if ESIflag = 1): entropic sensitivity indices
    o TURN-PHENO.bed (if MCflag = 1): turnover ratios
    o CAP-PHENO.bed (if MCflag = 1): channel capacities
    o RDE-PHENO.bed (if MCflag = 1): relative dissipated energies
    The methylation analysis files will be assumed to be located in INTERDIR, and the 
    output files will be stored in FINALDIR. These directories can be modified through
    optional arguments that can be passed to the function.

Usage:
    singleMethAnalysisToBed.sh [options] -- PREFIX

Mandatory argument:
    o PREFIX: prefix of the methylation analysis file (name phenotype).

Options:
    -h|--help           Help.
    -r|--refdir         Reference genome directory. Default: $REFGENEDIR.
    -a|--analdir        Analysis directory. Default: $REFGENEDIR.
    -d|--outdir         Output directory. Default: $FINALDIR.
    -t|--threshold      Threshold used in the methylation analysis. Default: 0.4.
    --min_chr           Minimum chromosome to be included in BED file. Default: 1.
    --max_chr           Maximum chromosome to be included in BED file. Default: 22.
    --MC                Computation of information theory metrics. Default: 0.
    --ESI               Computation of entropic sensivity index (ESI). Default: 0.
    -l|--MATLICENSE     Path to MATLAB's License.

Example:
    singleMethAnalysisToBed.sh -d out --min_chr 1 --max_chr 3 --MC --ESI -- normal

Output:
    BED files

Dependancies:
    MATLAB

Upstream:
